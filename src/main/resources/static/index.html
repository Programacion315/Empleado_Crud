Registro de Empleados

Esta aplicación tiene como fin, actualizar, buscar , crear y eliminar una lista de empleados. Para la persistencia como sistema gestor de base de datos utilice MySql y para administrar las bases de datos phpMyAdmin. Su Back End fue desarrollado en Java con el framework de Spring boot (Aca falta explicar mas). En la parte del Front End, use Html, Css, Js y la libreria bootstrap.
